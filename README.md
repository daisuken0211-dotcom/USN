# 半側空間無視・物体無視 体験（Webカメラ）

## できること
- カメラ映像を表示
- **画面左側（空間）** にランダムな「曇り/ぼかし/微小ズレ/ノイズ」を適用
- **検出された物体（COCO-SSD: 80カテゴリ）ごとに、その左半分（物体中心）**へランダムな「消失/曇り/粗視化/ぼかし」を適用

※ 教育・研修用の体験ツールです。診断・治療目的ではありません。

## 使い方（ローカル）
ブラウザのカメラは `https` または `localhost` が必要です。

### 1) このフォルダで簡易サーバーを起動
macOS / Linux:
```bash
python3 -m http.server 8000
```

Windows:
```bat
py -m http.server 8000
```

### 2) ブラウザで開く
- `http://localhost:8000/`

### 3) 「カメラ開始」
権限を許可してください。

## 使い方（配布/院内LAN）
- 可能なら `https` でホスト（GitHub Pages / 社内Webサーバーなど）
- 外部送信は行わず、ブラウザ内で推論します（CDNからライブラリは読み込みます）

## 対象クラスの絞り込み
UIの「対象クラス」へ `tv,cup,bowl,cell phone,remote` のように入力。
空欄なら全検出物体が対象です。

## 依存
- TensorFlow.js (CDN)
- COCO-SSD (CDN)

院内ネットでCDNが使えない場合は、依存JSをローカル同梱する版にもできます。
